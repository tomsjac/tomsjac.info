## Démarrage du projet

### Installation

#### 1/ Cloner le projet

A partir de la branche : `main-2_x`

```
git clone --branch main-2_x https://github.com/tomsjac/tomsjac.info.git tomsjac
```

#### 2/ Initialisation

Lancer les commandes
```bash
npm install
npm run dev
```

Allez sur l'url 
```
http://localhost:3000
```
